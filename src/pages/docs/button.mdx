# Button

Buttons allow users to trigger an action or event with a single click. For example, you can use a button 
for allowing the functionality of submitting a form, opening a dialog, canceling an action, or performing 
a delete operation.

## Variants

### Button with button tag

View example:

<div class="component-container horizontal">
    <Button tag="button">Button</Button>
    <Button tag="button" type='alternative'>Button</Button>
    <Button tag="button" type='ghost'>Button</Button>
</div>

HTML code:

```html
<button class="button button--primary">Button</button>
<button class="button button--alternative">Button</button>
<button class="button button--ghost">Button</button>
```

### Button with anchor tag

View example:

<div class="component-container horizontal">
    <Button tag="a" href="#">Button</Button>
    <Button tag="a" href="#" type='alternative'>Button</Button>
    <Button tag="a" href="#" type='ghost'>Button</Button>
</div>

HTML code:

```html
<a href="#" class="button button--primary">Button</a>
<a href="#" class="button button--alternative">Button</a>
<a href="#" class="button button--ghost">Button</a>
```
